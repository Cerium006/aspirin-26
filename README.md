Галактическая Эпидемия: Симулятор Лаборатории
Вы управляете секретной лабораторией. Ваша задача — создать лекарство от вируса "Aspirin-26" до того, как человечество вымрет.
Функционал:
- ООП Архитектура: Логика разделена на классы.
- GUI: Интуитивно понятный интерфейс управления.
- Графики (Matplotlib): Визуализация динамики заражения, смертности и выздоровления в реальном времени.
- Случайные события: Вирус может мутировать, увеличивая свою заразность или летальность.
- Экономика: Управляйте бюджетом, нанимайте ученых и улучшайте оборудование.
- Сохранение: Возможность сохранить текущий прогресс и рейтинг в файл `save_game.json`.

Как запустить локально
- Убедитесь, что у вас установлен Python 3.
- Установите необходимые библиотеки:
```bash
pip install matplotlib numpy Pillow pygame
```
- Запустите основной скрипт:
```bash
python main.py
```
Файлы проекта
- `main.py` — основной код приложения.
- `requirements.txt` — список зависимостей.
- `save_game.json` — файл сохранения (создается при нажатии кнопки "Сохранить").
